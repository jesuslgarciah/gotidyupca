# GoTidyUp

Go Tidy Up website built with Vite + React + TypeScript + Tailwind.

## Requirements

- Node.js 18+
- npm

## Local development

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
```

## Deploy to GitHub Pages

```sh
npm run build
npm run deploy
```
